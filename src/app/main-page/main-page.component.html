MoE アナウンス支援君(仮)Ver 0.0.3

mznh / moe / announce

α版です。予告なく 提供を終了する場合があります。by @mznh_moe

レイドやイベントのアナウンスがしやすくなる便利アプリです。アナウンス原稿の管理とゲームチャットへのコピペが簡単に行えます。

機能開発や公開継続の判断に使うので「イベントで使ったよ！」とか「いいね！」って思った方はtwitterのリプかマシュマロにて伝えてくれると嬉しいです。

要望やバグ報告も大歓迎。マシュマロはこちら！

変更点

  • 事あるごとにセーブするように変更（それに伴ってセーブロードボタン削除
  • イベント単位でのファイル出力と読み込み機能追加、読み込んだイベントは一番端タブに追加されます。
  • メッセージの並び替え機能追加
